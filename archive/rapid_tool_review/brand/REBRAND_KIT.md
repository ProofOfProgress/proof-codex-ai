# Rapid Tool Review — rebrand kit (owner steps)

**Do these in order on your laptop.** ~20 minutes total. No coding required.

---

## 1. Claim the name (YouTube Studio)

1. Open [YouTube Studio](https://studio.youtube.com) → **Customization** → **Basic info**
2. **Name:** `Rapid Tool Review`
3. **Handle:** `@RapidToolReview` (grab before someone else does)
4. **Description:** copy everything under `DESCRIPTION:` from `channel/brand/youtube_copy.txt`
5. **Save**

---

## 2. Upload channel art

PNG files are in the repo (or generate fresh):

```bash
python3 -m shorts_bot.brand.assets_cli
```

| Asset | File | Size |
|-------|------|------|
| Profile picture | `channel/brand/assets/profile.png` | 800×800 |
| Banner | `channel/brand/assets/banner.png` | 2560×1440 |

Studio → **Customization** → **Branding** → upload both.

---

## 3. Pinned comment (on your next Short)

```
What AI tool should I review next? Name one — Pay, Skip, or Wait in 30 seconds.
```

---

## 4. Video title format (every Short)

```
[Product] — [Verdict hint]? | Rapid Tool Review
```

Examples:
- `ChatGPT Plus — Skip? | Rapid Tool Review`
- `InVideo AI — Worth It? | Rapid Tool Review`
- `Claude Pro — Pay or Skip? | Rapid Tool Review`

---

## 5. Description template (paste per video)

```
Honest 30-second AI tool review — Pay, Skip, or Wait.

🔧 Tool: [PRODUCT NAME]
✅ Verdict: [PAY / SKIP / WAIT]

Subscribe for daily AI tool verdicts: @RapidToolReview

#aitools #techshorts #rapidtoolreview
```
