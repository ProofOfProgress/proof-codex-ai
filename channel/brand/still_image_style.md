# Still image style — Soft Continuity (timestamp slideshow Shorts)

Used when generating **one image per TurboScribe timestamp** for CapCut timeline edit.

## Visual rules (9:16 still frame)

- Faceless. No real people, no celebrity likeness.
- Calm self-help mood: sleep, focus, boundaries, quiet rooms, soft light.
- Subtle oracle beat: one symbolic element max (thin ring, faint glow, window light) — never horror, never explicit AI/robot.
- Minimal composition. Lots of negative space. One clear subject.
- Color: deep navy `#0B1020`, mist `#8EB8FF`, warm text accent `#F2F5FA`.
- **Not** MS Paint scribble style — Soft Continuity is polished-minimal, not “bad drawing” meme.

## Image prompt template (per timestamp)

Each image illustrates **only the words spoken from that timestamp until the next**.

Include in every prompt:
- "vertical 9:16 still image, no text, no watermark"
- "faceless, calm, soft continuity aesthetic"
- scene description from that script slice

## Edit rhythm

- Image changes every 2–8 seconds (whatever TurboScribe timestamps give).
- Filename = start second → drag on CapCut timeline until next filename second.
