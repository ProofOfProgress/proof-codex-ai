# Brand voice — inject into all drafts

When writing hooks and scripts for this channel:

1. **Help first** — viewer must leave with one concrete action.
2. **Warm register** — like a patient friend, not a coach screaming.
3. **One oracle line** — max one sentence per script that feels subtly too knowing.
   - Pattern: "This is older than you think." / "It works on the days you don't believe it."
4. **No AI words** — never say bot, algorithm, ChatGPT, simulation, glitch.
5. **Mute-safe** — on-screen text carries the oracle beat if voice is too subtle.
6. **End clean** — payoff lands, then stop. No doom linger.

Tagline options for CTA slide: "You're still here." / "Good. Stay."
