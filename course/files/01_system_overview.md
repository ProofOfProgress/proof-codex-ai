# 01 — System Overview and Decision Ladder

Use when the request is broad, messy, or spans multiple levers. Triage when the user says "help with my content" or "why aren't my videos working."

## Semantic anchors
idea, hook, visuals, retention, payoff, rewatchability, comments, CTA, analytics, short-form storytelling, virality

## The system in one view
Jenny's system starts upstream. Idea comes before polish. If the idea is weak, better production does not rescue it.

Working chain: idea → hook → visual clarity → momentum → payoff → rewatch/comment/community → CTA/conversion → analytics diagnosis

## Default decision ladder
1. Check the idea first.
2. Check whether the idea can be hooked immediately.
3. Check whether the story is visually clear without sound.
4. Check whether every beat pushes toward payoff.
5. Check whether the payoff lands clearly and ends promptly.
6. Check whether rewatchability and comment design are intentional.
7. Check CTA placement only after the content engine works.
8. Use analytics after publishing to diagnose where the system broke.

## Route elsewhere when
- Premise strength → 02
- Stand out / positioning → 03
- New ideas / remixes → 04
- Props, visuals, mute clarity → 05
- Structure, pacing, payoff → 06
- Relatability, evergreen, comments → 07
- Editing or music → 08
- Growth, CTA, selling, analytics → 09

## Jenny-approved filter
- Do not overprioritize cinematic polish over clarity.
- Do not add dead space or irrelevant tangents.
- Do not put the CTA only after the payoff.
- Do not overload with music switches that make the video feel finished halfway through.
