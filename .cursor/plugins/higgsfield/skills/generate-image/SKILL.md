# Generate image

Use the Higgsfield MCP `generate_image` tool. For local file references, upload via `media_upload` + `media_confirm` first.

## Examples

```
Generate a cinematic poster of a lone astronaut on a red dune at sunset, 16:9, photorealistic
Three variations of this character in a noir style
Square YouTube thumbnail about quantum computing — bold, high contrast
```
