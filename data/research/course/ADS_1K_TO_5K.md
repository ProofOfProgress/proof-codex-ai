# Follow ads growth (1k → 5k followers)

**Status:** Saved for later — **not active now.** Run this when an account hits **~1,000 followers** (after bubble wrap or equivalent warming).

**Goal:** Use a viral “journey” mirror video + TikTok Ads Manager **follow** campaign to reach **5,000 followers**, then stop the ad.

---

## Quick checklist

1. **Find the format** — TikTok search (verbatim): `im so exited to start my journey with this app`  
2. **Film the same video** — mirror, phone in hand, same caption + “Who’s with me?” mid-video  
3. **Sound** — use the sound all those videos use (**“Her” by The American Dream** — **owner must confirm** when it’s time)  
4. **Post** on the account you want to grow (don’t overthink it)  
5. **TikTok for Business** (desktop) → create/login business account  
6. **Ads Manager** → Campaigns → Create → objective **Brand awareness and community interaction**  
7. Name **follow ad** · optimization **Follow** · connect the posting account · add that video  
8. Targeting: custom · **all countries except South Korea** · language/gender **all** · **$20–30/day** · continuous schedule  
9. Launch **next day 12:01 AM** · Publish · wait for approval  
10. **Stop the campaign** as soon as you hit **5k followers**

## Step 1 — The video (organic post first)

### Find examples

In TikTok search, type **exactly**:

```
im so exited to start my journey with this app
```

*(Spelling matches the viral trend — “exited” not “excited”.)*

You’ll see many similar clips: person in a **mirror**, **holding their phone** while recording.

### Copy the format

| Element | What to do |
|--------|------------|
| **Shot** | Mirror selfie, phone visible, same vibe as top results |
| **Caption / on-screen** | `im so exited to start my journey with this app` |
| **Mid-video text** | **Who’s with me?** (pops up in the middle) |
| **Sound** | Same sound every viral video uses — course says **“Her” by The American Dream** |

**Owner rule:** When it’s time to run this, **you double-check the sound** on TikTok before posting (trend sound can change).

Post on the account you’re growing. Keep it simple — match what’s already working in search.

---

## Step 2 — TikTok Ads Manager setup

All on **computer**:

1. Go to **TikTok for Business**  
2. Create a **business account** if you don’t have one  
3. Open **TikTok Ads Manager** and log in with that business account  
4. **Campaigns** → **Create campaign**

### Campaign settings

| Field | Value |
|-------|--------|
| **Advertising objective** | Brand awareness and community interaction |
| **Campaign name** | `follow ad` |
| **Special categories** | None |
| **Optimization goal** | **Follow** |
| **Connected account** | The TikTok account that posted the journey video |
| **Creative** | Add **that most recent video** (Add videos and images) |

### Targeting & budget

| Field | Value |
|-------|--------|
| **Targeting** | Custom |
| **Location** | **Every country except South Korea** |
| **Language** | All |
| **Gender** | All |
| **Budget** | Custom — **$20–30 per day** |
| **Schedule** | Custom — **run continuously** (no end date in Ads Manager) |

### Launch

- Schedule start: **next day at 12:01 AM** (relative to when you set it up)  
- **Publish** → wait for **approval**  
- Let it run until the account hits **5,000 followers**  
- **End the campaign immediately** at 5k — don’t leave it running
