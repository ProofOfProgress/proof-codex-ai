# Violation appeals — emergency fallback only

**Owner override (north star):** The goal is **never get a strike at all** — Module 1 + TOS QC before every upload (`TIKTOK_SHOP_CONTENT_POLICY.md`). This file is **not** the strategy. Use it only if TikTok flags a clip we already passed QC (rare false positive).

When an account or video gets hit — **human + ChatGPT step**, not fully automated by the bot.

## Quick steps

1. Screenshot the **violation notice** in TikTok  
2. Open **Momentum Academy Notion → Custom Violation Appeal Prompts**  
3. Pick **Prompt 1, 2, or 3** (see `module_08_beating_violations.md`)  
4. **ChatGPT:** attach violation screenshot + paste full Notion prompt  
5. Submit appeal text (**≤ 500 characters**) + attach screenshots ChatGPT lists  
6. Check **Discord `#violation-appeals`** if violation type matches a forum post  
7. Stuck → **1-on-1 coaching ticket**, tag coaches  

## Violation waves

Wait for **group call / Discord / owner message** — log in `GROUP_CALLS.md`. Don’t guess during a wave.

## Bot role

- **Primary job:** prevent strikes — Module 1 + TOS QC blocks upload until zero violations  
- **Do not:** auto-submit TikTok appeals (needs owner screenshots + ChatGPT)  
- **Do not:** lower QC standards to hit 8–10/day and rely on appeals afterward  

When owner pastes **Prompt 1** and **Prompt 2** full Notion text, save here for agent use.
