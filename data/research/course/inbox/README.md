# Course inbox — recordings & coach drops

Drop files here so the cloud agent can find them on the hub or in git.

## Voice recordings (hub laptop)

1. Record with **Windows Voice Recorder**
2. Rename before or after: **`coach-call-YYYY-MM-DD.m4a`**
3. Default save folder: `C:\Users\isaac\OneDrive\Documents\Sound recordings\` (OneDrive sync; legacy path without OneDrive also works)
4. Optional copy to WSL: `~/proof-codex-ai/data/research/course/inbox/`

After the call, message the agent:

> Transcribe **coach-call-2026-06-30** and update the bot

## Discord / coach samples

- Paste links or export notes into `coach-call-YYYY-MM-DD.md` in this folder
- Screenshot filenames: `discord-good-example.png`, `discord-bad-example.png`

## Hub must be on

Double-click **START HUB** on the laptop Desktop so the agent can SSH in and grab the file.
