## Step 3 — Make the image (ChatGPT)

Do **not** post the same screenshot as everyone else. **Copy the format, change one detail.**

1. Find a **viral slide** on the sound page (e.g. bubble-wrap bathroom, bubble duck, bubble suit).  
2. **Screenshot** the image.  
3. Open **ChatGPT** → attach screenshot → prompt:

```
Make me an image exactly like this but change [one small detail].
```

**Examples from the video:**

- Bubble duck in tub → **bubble dog** instead  
- Bubble-wrap bathroom → same layout, swap one labeled object  
- Engagement slide (Trump bubble suit) → same **Pause / Comment / Share / Follow** text layout, different subject  

4. Download the generated image — **unique to this account**.

**Rules:**

- **No stock photos** — AI-generated only  
- Change **one small thing** so it’s not a duplicate  
- Takes ~10 seconds–2 minutes per image once you have the workflow

## Posting volume & multi-account strategy

| Tactic | Detail |
|--------|--------|
| **Accounts** | Run **3–5 accounts** on this strategy in parallel |
| **Uniqueness** | **Never** post the **same image** on multiple accounts — each account gets **unique** videos |
| **Volume mix** | Some accounts: **1 video/day**. Others: **5–10 videos/day** |
| **Goal** | Post until **1,000 followers** |
| **Luck** | One video can go mega-viral and jump you to **500–600+** followers in a hit |

Then switch focus to **affiliate product content** (Modules 3–8) on warmed accounts.
