## 7 steps (his exact order)

1. **Sign up** — TikTok Seller Center (individual seller, US ID + SSN + bank)
2. **Printify** — connect to TikTok Shop (POD default for beginners)
3. **Link TikTok** — seller-operated official account (0 followers OK)
4. **Winning product** — copy/twist proven viral designs (TikTok search first, Kalodata validate)
5. **Order your own sample** — from your live listing, not direct from Printify
6. **In-house content** — 3–5 videos/day warm the listing before heavy affiliate push
7. **Affiliates + GMV Max** — onboard creators, then ads on winners

## Seller setup checklist (you)

1. Go to https://seller-us.tiktok.com → **Individual seller**
2. Email login (not Google) — easier recovery
3. ID + SSN + bank + broad shop name (ChatGPT OK for name)
4. **Do not** set personal address as warehouse if using Printify
5. Connect **Printify Premium** → pickup warehouse: Printify Fort Worth address (see transcript)
6. Seller Center → link **official TikTok account** (QR code from phone app)
7. Confirm: shop icon on TikTok + can tag products in videos

**Working capital:** ~$1,000 float for POD (Printify charges before TikTok pays you).

## Product research (his method)

1. **Discover on TikTok** — search + shop tab (not Amazon/Etsy first)
2. **Validate in Kalodata** — 30/180-day sales, creators, top videos
3. **Copy or twist** proven winners:
   - 1,000+ units sold + few reviews = early, copyable
   - Old winner with deep review history = twist/dupe, don't clone
4. **POD niches that work:** funny phrase shirts, giftable (Valentine's, Mother's Day), political humor
5. **Design rules:** short front text, 3–5 colors max, first 3 mockup images matter most

**EchoTik note:** Our bot uses EchoTik instead of Kalodata — same logic (discover → validate → score), different tool.

## Affiliate program (how he gets others to post for you)

| Type | What it is |
|------|------------|
| **Open collaboration** | Default commission visible to all affiliates (~20%) |
| **Target collaboration** | Direct invite to specific creators (preferred) |

**Commission:** start **~30%** for new shops (not stingy). Message creators in **$/order**, not %.

**Onboarding SOP:**
1. Find creators on competitor videos (Kalodata) or Find Creators tab
2. Send **target invite + TikTok DM** (DM matters — many ignore invites)
3. **Manual** sample approval — vet content quality first, not GMV alone
4. KPI: **videos per sample sent** (goal 2–5+, top ops get 7–20)
5. 4–5 test videos before any retainer
6. Move winners to WhatsApp/text + weekly **Notion creator brief**

**Retainer market:** ~30 videos / $1,000/mo — he prefers **16–20 videos / $400** + boosted commission.

## GMV Max ads (when ready)

- Start when **25–50 videos/week** in circulation OR clear organic winners
- Campaign: **Promote products** (not LIVE)
- Example: **$200/day**, target ROI **~5** (POD t-shirt example)
- Break-even ROI ~2.4 on $24 shirt — use ChatGPT for your SKU math
- Creative dies in ~2–3 weeks — need constant fresh affiliate + in-house volume

---

## Violations

- TikTok uses **AI** for violations — fight with **AI-written appeals**
- First appeal = AI review; second = human
- Use Seller Center → Account health → live chat → expedited ticket
- Creator brief must include **claims dos/don'ts** (especially supplements)

---

## Red flags (don't do)

- Auto-approve affiliate samples
- Send samples before warming listing with your own videos
- Low commission at launch
- Retainer on day one
- Edit listing in TikTok when using Printify (edit in Printify only)
- GMV Max before enough creative volume
- Standard dropship (2-day ship rule) — POD/Printify exception
- Buy 5K follower account for affiliate path unless you accept ban risk
