# TikTok Shop strategy — LOCKED (2026-06-24)

**Owner decision:** Sell **physical gadgets** on **TikTok Shop**. Product-hop constantly. **Fix It Fast** brand.

---

## Hierarchy

```
Home / car / kitchen gadgets
    └── Problem-solver impulse buys ($15–$35)
            └── One Short = one product = one problem fixed
```

**One-line pitch:** Annoying problem → satisfying demo → orange cart.

---

## Business model

1. **List or source products** in TikTok Shop (catalog / supplier / dropship)
2. **Post Short** — problem hook, hands demo, Shop CTA
3. **Track sales** — winner gets more angles; loser gets dropped
4. **Repeat** — hop to next SKU (infinite test queue)

**Not this channel:** AI software affiliates, Rapid Tool Review (retired), inventory-heavy custom merch day one.

---

## Video format

| Beat | What happens |
|------|----------------|
| Hook (0–2s) | Show the problem |
| Demo (2–18s) | Product fixes it |
| Payoff (18–24s) | One benefit line |
| CTA (24–28s) | Orange cart / shopping bag |

**Rules:** One product per Short. No listicles. No fake medical claims. Honest demo.

---

## Production

```
Product queue (data/product_queue.json)
    → Shop brief (InVideo — hands + product, no twin)
    → MP4 → TikTok upload (+ optional YouTube)
    → Sales analytics → kill losers / scale winners
```

InVideo: Basic ≤10 credits. NO AI Twin. NO Ms. Byte.
